🔍 Overview
This project focuses on cleaning and analyzing payroll data using Power BI and Microsoft Copilot to generate meaningful financial insights.

🎯 Objectives:
Clean and preprocess payroll data
Create KPIs using DAX
Build interactive dashboard
Use Copilot for AI-based insights

Problem:
Payroll contains errors like duplicates,incorrect formats,and unstructured values,making analysis difficult 

Solution:
Data Cleaning and transformation, 
KPI creation in Power BI,
Dashboard Visualisation ,
AI integration using Copilot. 

Tools Used:
Power BI ,
Excel,
Microsoft Copilot ,
Kaggle Dataset
